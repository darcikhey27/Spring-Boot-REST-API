Darci Tiny Weather Dashboard API

Before getting any weather information from a city, a city has to added to the database with add api (POST call)

use POST to add a city to the database, then use GET to get weather info for that city that you just entered

Weather Widget API

POST

For adding a city to the database

darcikhey.com/api/v1/add?city={city_name}

$.ajax({
    type: 'POST',
    url: '/api/v1/weather/city/add/',
    data: { city_name: name },
    success: function (data) {
        // do something with data
    }
});

GET

For getting a city by name

darcikhey.com/api/v1/weather/get?city_name={city_name}

For getting a city by id

darcikhey.com/api/v1/weather/get?city_id={id_number}

For getting a all cities in database

darcikhey.com/api/v1/weather/get?city_all=all

$.ajax({
    type: 'GET',
    url: '/api/v1/weather/get?city_name='+name,
    success: function (data) {
        // use data
    }
});

UPDATE

For updating a city weather information by city name

darcikhey.com/api/v1/update?city={city_name}

For updating a city weather information by city id

darcikhey.com/api/v1/update?city_id={id_number}}

To update all cities in database

darcikhey.com/api/v1/update?city=all

DELETE

To delete a city by name

darcikhey.com/api/v1/delete?city={city_name}

To Delete a city by using city_id

darcikhey.com/api/v1/update?city_id={id_number}}

To delete all cities in database

darcikhey.com/api/v1/delete?city=all
